# Dev Tools — Overview

Shortcuts to runtime skill packages and the Codex playbook skills that cover them.

## Playbook index (type-based)
- [.codex/skills/README.md](../../.codex/skills/README.md) — Full index of 15 types and their configurations.

## Types relevant to dev-tools
- **Runtime skills (run, patch, maintain):** Use the **repo** type — configurations `dev-tools-run-skill`, `dev-tools-patches`, `dev-tools-skill-maintenance`, plus `tool-definitions`, `contracts`, `run-commands`, etc.
- **TypeScript/JavaScript:** Use the **code** type — choose workflow (debug, implement, quality, refactor, testing, lint, codegen, …) and language TS.
- **PowerShell:** Use the **code** type — same workflows, language PS.
- **Core guardrails:** Use the **core** type — debug-root-cause, tdd-red-green, verify-before-claim.
